# 🔍 VulnScan Pro — Network Vulnerability Scanner

An interactive browser-based network vulnerability scanner built to simulate real-world penetration testing workflows. Designed as a portfolio project by **Prajwal Shelke**, an aspiring Application Security professional.

## Features
- **4 Scan Modes** — Quick, Full Scan, Stealth (SYN), and Deep Exploit
- **Live Terminal** — Real-time scan log with color-coded output
- **Port Discovery** — Maps open, closed, and filtered ports with service detection
- **CVE Mapping** — Correlates findings with real CVE IDs and CVSS scores
- **OWASP Classification** — Tags each vulnerability with OWASP Top 10 category
- **Risk Score Meter** — Composite threat scoring (0–100)
- **Auto Report Generation** — Professional penetration test report with executive summary
